# VisitorLog
A free and open source visitor logging system with the following technologies:
- Frontend: Svelte
- REST API: .Net Core & ServiceStack
- Database: MongoDB via MongoDB.Entities

# How to run:
```
cd docker
docker-compose up -d
```

# Addresses:
### Web UI
> http://localhost:8080

### API Server
> https://localhost/*

### MongoDB Server
> mongodb://localhost:27017
